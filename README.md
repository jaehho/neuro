# neuro

Neuromodulated STDP simulation: pre-synaptic neurons → 1 post-synaptic LIF neuron with three-factor learning rules (Frémaux & Gerstner 2016).

See `CLAUDE.md` for project structure and commands. Open `notebooks/verified.py` with `uv run marimo edit` for the focused 1-pre + target-rate + covariance path; `notebooks/exploratory.py` covers everything else.
